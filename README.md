# FutuRaPS Adaptive Spraying

This repository contains four ROS 2 packages developed in connection with a
master's thesis on robotic adaptive spraying:

- `futuraps_perception`
- `futuraps_task_planner`
- `futuraps_platform_pid`
- `futuraps_spray_coverage`

The repository is currently under development. Further documentation,
installation instructions, and usage examples will be added later.
